RimWorld AI Translator 패키지

실행:
1. RimWorldAiTranslator.exe 를 더블클릭합니다.
2. 모드 화면에서 기존 작업을 열거나, 감지된 모드를 불러옵니다.
3. 검색되지 않는 모드는 폴더 선택으로 직접 지정합니다.
4. 설정 탭에서 Cerebras API 키가 있으면 한 줄에 하나씩 입력합니다. 비워두면 Google 번역으로 후보를 만듭니다.
5. 배치 크기는 안정성을 위해 40으로 고정됩니다.
6. 실행 결과는 항상 검수 폴더에 먼저 생성됩니다.
7. 번역이 끝나면 같은 화면에서 파일별 문자열, 원문, 기존 번역, 번역 후보, 용어, 메모, 경고를 확인합니다.
8. 목록 위의 상태 버튼으로 전체, 미번역, 번역됨, 검토됨, 변경됨 항목을 즉시 필터링할 수 있습니다.
9. 항목 상태는 미번역, 번역됨, 검토됨으로 관리하며, 업데이트로 원문이 바뀐 항목은 별도 표식과 필터로 표시합니다.
10. 검토 적용 버튼은 검토 완료 항목만 적용하고, 전체 적용 버튼은 안전한 번역 후보까지 함께 적용합니다.
11. 번역칸은 AI 후보, 기존 한국어 번역 순서로 기본값을 채우며 둘 다 없으면 직접 입력할 수 있습니다.

필요 환경:
- Windows 10/11
- Windows 기본 PowerShell
- 인터넷 연결
- Cerebras API 키는 선택 사항입니다. 없으면 Google 번역 fallback을 사용합니다.

모드 목록:
- Steam libraryfolders.vdf, SteamLibrary, RimWorld 설치 폴더를 자동 검색합니다.
- 감지 결과는 모드 루트가 바뀌지 않은 동안 로컬 캐시를 재사용하며, 새로고침 버튼은 전체 목록을 다시 검색합니다.
- 목록에서 모드를 클릭하면 해당 모드의 작업 상태가 저장됩니다.
- 검색되지 않는 모드는 폴더 선택 버튼으로 직접 지정할 수 있습니다.

추가 용어집 TXT 형식:
원문=번역
원문 => 번역
원문<Tab>번역<Tab>메모

기본 용어집:
- glossary.generated.ko.json
- 림월드 본편과 공식 DLC 기준 용어집입니다.
- RMK/모드 용어는 기본 포함하지 않습니다.

주의:
- Steam 워크샵 원본 모드 폴더에 직접 쓰면 Steam 업데이트로 덮어써질 수 있습니다.
- API 키는 화면 공유나 스크린샷에 노출되지 않게 주의하세요.
- GUI에 입력한 API 키는 파일이나 명령줄에 저장되지 않지만, 번역 원문은 선택한 Cerebras 또는 Google 외부 서비스로 전송됩니다.
- 이 오픈 프로젝트의 EXE와 PowerShell 스크립트는 상용 코드 서명 인증서로 서명되지 않았으므로 Windows에서 게시자 경고가 나타날 수 있습니다.
- 검토됨 적용은 검수 상태 파일의 검토됨 항목만 적용합니다. 번역됨 적용은 검토됨과 안전한 번역됨 항목을 함께 적용합니다.
- 업데이트로 같은 키의 원문이 바뀌면 이전 원문을 기록에 보존하고 미번역 상태로 내리며, 다시 처리할 때까지 적용 대상에서 제외합니다.
- Google 번역 fallback은 추가 프롬프트와 용어집을 반영하지 못하므로 검수 창에서 확인하는 것을 권장합니다.
- 설정에서 시스템/밝은/어두운 테마, 본문 글자 크기, 고대비, 자동 저장 여부를 바꿀 수 있습니다. API 키는 설정 파일에 저장하지 않습니다.
- 검수 창 단축키: Ctrl+F 검색, Ctrl+S 저장, Ctrl+Enter 검토 후 다음, F6 영역 이동, Alt+위/아래 이전/다음 항목.
- DLL/C# 코드에 직접 박힌 inspect 패널/gizmo/상태 문구는 XML 번역만으로 바뀌지 않을 수 있습니다. 모드가 번역 키를 쓰지 않는 경우 원본 코드 수정이나 Harmony 패치가 필요합니다.
