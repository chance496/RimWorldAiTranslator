RimWorld AI Translator
======================

RimWorld 모드를 프로젝트별로 불러와 AI 초벌 번역, 수동 번역, 검수,
업데이트 추적, Languages\Korean 적용과 RMK 로컬 작업까지 처리하는 Windows용 도구입니다.

실행
----
1. RimWorldAiTranslator.exe를 실행합니다.
2. 자동 검색된 모드를 선택하거나 모드 폴더를 직접 지정합니다.
3. 원문 언어가 여러 개면 프로젝트에서 사용할 기준 언어를 선택합니다.
4. 프로젝트를 열고 원문 갱신을 누릅니다.
5. Cerebras API 키를 한 줄에 하나씩 입력한 뒤 AI 번역을 누릅니다.
6. API 키가 없으면 Google 번역 후보를 생성합니다.
7. 기존 번역이 있으면 덮어쓰기, 미번역 부분만 번역하기 또는 취소를 선택합니다.
8. 번역문을 확인하고 번역됨 또는 검토 완료 상태로 지정합니다.
9. RMK에 적용을 끄면 원본 모드에, 켜면 RMK 작업 클론에 기록합니다.
10. 검토 적용은 검토됨만, 전체 적용은 번역됨과 검토됨을 함께 반영합니다.

주요 기능
---------
- 프로젝트 하나당 모드 하나를 연결합니다.
- Steam Workshop과 로컬 RimWorld 모드를 자동으로 검색합니다.
- 원본 언어를 감지하고 여러 언어가 있으면 프로젝트 생성 때 기준 언어를 선택합니다.
- 여러 Cerebras API 키를 입력 순서대로 순환 사용합니다.
- 기존 번역을 전체 재번역하거나 미번역 부분만 번역할 수 있습니다.
- 미번역, 번역됨, 검토됨, 변경됨 상태와 텍스트, 키, Def Class, Node 검색을 제공합니다.
- 같은 원문을 수동 번역하면 동일 원문 전체에 적용할지 확인합니다.
- 원문 갱신 후에도 모드에 아직 적용하지 않은 번역과 메모를 유지합니다.
- 같은 키의 원문이 바뀌면 기존 번역을 보존하고 변경됨 + 미번역으로 표시합니다.
- RMK 구독본의 기존 번역을 재사용하고, 없는 키만 AI 번역할 수 있습니다.

상태와 적용
-----------
- 미번역: 번역이 없거나 업데이트 후 다시 확인해야 합니다.
- 번역됨: 번역은 있지만 최종 검토 전입니다.
- 검토됨: 사람이 확인했습니다.
- 변경됨: 같은 키의 원문 값이 바뀌었습니다. 다시 처리하기 전에는 적용되지 않습니다.

검토 적용은 검토됨 항목만 기록합니다.
전체 적용은 검토됨과 안전한 번역됨 항목을 함께 기록합니다.

RMK 로컬 연동
-------------
- Steam Workshop RMK 구독본은 기존 번역 참조에만 사용하며 수정하지 않습니다.
- RMK에 적용을 체크하지 않으면 기존처럼 원본 모드의 Languages\Korean에 씁니다.
- 체크하면 설정에서 지정한 RMK bus 브랜치 작업 클론에 키 기준으로 병합합니다.
- 기존 키는 원래 RMK XML에서 수정하고 새 키만 추가한 뒤 LoadFoldersBuilder를 실행합니다.
- 원문 변경 또는 안전 검사 실패 항목은 내보내지 않습니다.
- Git 커밋과 푸시는 자동으로 하지 않습니다.

동일 원문 일괄 번역
-------------------
직접 입력한 번역을 저장하거나 다른 항목으로 이동할 때 같은 원문이 있으면
일괄 적용 여부를 확인합니다. 줄바꿈을 제외하고 원문이 완전히 같을 때만 묶습니다.
메모는 복사하지 않으며, 바뀐 다른 항목은 번역됨 상태로 돌아가 다시 검토할 수 있습니다.

주요 단축키
-----------
F2 번역문 입력 / F3 다음 / Shift+F3 이전
Alt+1 AI 후보 / Alt+2 기존 번역 / Alt+0 되돌리기
Ctrl+1 미번역 / Ctrl+2 번역됨 / Ctrl+3 검토 완료
Ctrl+Enter 검토 완료 후 다음 / Ctrl+F 검색 / Ctrl+S 저장
F5 원문 갱신 / F9 AI 번역 / Shift+F9 중지 / F6 작업 영역 이동

API와 개인정보
-------------
- 기본 모델: Cerebras gemma-4-31b
- 기본 배치: 40개
- API 키는 파일, 프로젝트, 명령행, 로그에 저장하지 않습니다.
- GUI의 API 키는 번역 프로세스 환경 변수로만 전달하며 임시 인수 파일에도 넣지 않습니다.
- 번역 실행은 cmd.exe 셸을 거치지 않고 허용된 매개변수만 전달합니다.
- 프로그램을 다시 실행하면 API 키를 다시 입력해야 합니다.
- 번역 원문은 선택한 Cerebras 또는 Google 외부 서버로 전송됩니다.

로컬 데이터
-----------
프로젝트와 검수 데이터는 다음 위치에 저장됩니다.

%LOCALAPPDATA%\RimWorldAiTranslator

- projects.json: 프로젝트와 모드 연결 정보
- reviews\...\review-decisions.json: 번역문, 상태, 메모, 원문 변경 기록
- settings.json: 화면 설정

프로그램 ZIP이나 EXE를 업데이트해도 이 데이터는 유지됩니다.

요구 환경
---------
- Windows 10/11
- Windows 기본 PowerShell
- 네트워크 번역 사용 시 인터넷 연결
- 기본 기능에는 Python, Node.js, Git이 필요하지 않습니다.
- RMK에 적용하려면 RMK Git 클론과 Git이 필요합니다.

주의
----
- defName, XML 키, 클래스명, 텍스처 경로 같은 게임 식별자는 번역하지 않습니다.
- DLL/C# 코드에 직접 작성된 화면 문자열은 XML 번역만으로 바뀌지 않을 수 있습니다.
- 외부 번역 서비스에 보내면 안 되는 비공개 문자열에는 네트워크 번역을 사용하지 마십시오.
- 배포 EXE는 코드 서명 인증서로 서명되지 않아 Windows 경고가 나타날 수 있습니다.
