RimWorld AI Translator v1.1.0
================================

실행 방법
---------
1. ZIP 파일의 압축을 완전히 풉니다.
2. RimWorldAiTranslator.exe를 실행합니다.

Windows 10/11 64비트를 지원합니다.
별도의 .NET 설치는 필요하지 않습니다.

간단한 사용법
-------------
1. 모드 폴더를 선택하고 프로젝트를 만듭니다.
2. 설정에서 번역 서비스, 모델, API 키를 입력합니다.
3. AI 번역을 실행하고 결과를 검토합니다.
4. 검토 적용 또는 전체 적용으로 저장합니다.

알려진 문제
-----------
- 디지털 서명이 없어 Windows SmartScreen 경고가 표시될 수 있습니다.
- 무료 API 모델은 호출 제한이나 일시적인 장애가 있을 수 있습니다.
- API 키는 저장되지 않으므로 실행할 때마다 다시 입력해야 합니다.
