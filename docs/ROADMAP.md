## 자율 작업 시작 차단 요소

1. 현재 dirty 제품 변경의 안전한 로컬 체크포인트가 없다.
2. 회귀 테스트 harness가 없어 P0 적용 쓰기와 현재 Def/RMK 변경을 승인할 수 없다.
3. 최근 호스트 프리즈와 비정상 종료가 있어 패키지 빌드·GUI·대형 fixture 검증 전 시스템 안정 확인이 필요하다.
4. push/release 또는 외부 백업은 현재 요청 범위가 아니므로 별도 사용자 요청 전에는 수행할 수 없다.
