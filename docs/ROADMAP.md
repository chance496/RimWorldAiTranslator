# Roadmap

우선순위는 사용자 영향과 데이터 안전을 기준으로 한다. `READY`는 선행 조건이 충족된 항목, `BLOCKED`는 명시된 조건이 해결되기 전 시작하지 않는 항목이다.

## P3 - 선택 기능과 실험적 개선

2026-07-12 재평가: 기존 P3-01~P3-03은 각각 동작하지만 장시간 검수와 품질 관리에 필요한 통합 도구 묶음은 아니다. 아래 기능은 장식용 UI가 아니라 현재 프로젝트 데이터와 직접 연결한다.

| ID | 상태 | 작업 | 완료 조건 |

## 남은 외부 차단 요소

1. 최근 호스트 프리즈와 비정상 종료 이력이 있으므로 무거운 빌드·GUI 감사를 동시에 실행하지 않는다. 이번 P0-P3 전체 회귀, UI 감사, RMK benchmark와 패키지 빌드 중 재발 징후는 없었다.
2. push/release 또는 외부 백업은 현재 요청 범위가 아니므로 별도 사용자 요청 전에는 수행할 수 없다.

P0/P1 안정성은 완료 상태를 유지한다. P2/P3는 위 재평가 항목을 완료하고 전체 회귀·시각·성능 검증을 다시 통과할 때 최종 완료로 복귀한다. P2-03의 실제 125/150/200% DPI는 시스템 표시 배율 변경이 필요한 외부 검증으로 남는다.
