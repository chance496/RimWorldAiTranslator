# Autonomous Decisions

- 기준 백업 참조는 이동·삭제·push하지 않는다.
- 실제 사용자 데이터 대신 고유한 `%TEMP%` fixture와 격리 앱 데이터를 사용한다.
- 성능 작업은 기능을 제거하지 않고 동일 결과의 비용만 줄인다.
- 실제 125/150/200% DPI는 Windows 표시 설정 변경이 필요하므로 로컬 자동 작업에서 차단한다.
- push, release, 실제 API와 외부 업로드는 별도 사용자 요청 없이는 수행하지 않는다.

제품의 장기 설계 결정은 `../DECISIONS.md`가 권위 문서다.
