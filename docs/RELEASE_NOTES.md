# RimWorld AI Translator v1.1.0

## 프로그램 소개

RimWorld 모드의 원문 분석, 번역, 검토, 적용을 한곳에서 처리하는 Windows 프로그램입니다.

## 주요 기능

- 기존 프로젝트와 검토 데이터 불러오기
- 기본 RimWorld 용어집과 사용자 용어집 지원
- AI 번역 진행률과 재시도 상태 표시
- 여러 API 키 사용과 공급자 호출 제한 자동 대응
- 번역 중지 후 완료 결과와 미번역 원문 보존
- 창 이동·크기 변경 시 화면 깜빡임과 글자 잘림 개선
- 버튼 설명, 단축키, 비활성 사유 ToolTip 제공
- 로컬 모드 적용과 RMK 작업 폴더 내보내기

## 다운로드

`RimWorldAiTranslator-v1.1.0-win-x64.zip`을 다운로드하세요.

## 실행 방법

1. ZIP 파일의 압축을 완전히 풉니다.
2. `RimWorldAiTranslator.exe`를 실행합니다.

Windows 10/11 64비트를 지원하며 별도의 .NET 설치는 필요하지 않습니다.

## 간단한 사용법

1. 모드 폴더를 선택하고 프로젝트를 만듭니다.
2. 설정에서 번역 서비스, 모델, API 키를 입력합니다.
3. AI 번역을 실행하고 결과를 검토합니다.
4. 검토 적용 또는 전체 적용으로 저장합니다.

## 알려진 문제

- 디지털 서명이 없어 Windows SmartScreen 경고가 표시될 수 있습니다.
- 무료 API 모델은 호출 제한이나 일시적인 장애가 있을 수 있습니다.
- 공급자 정책 변경으로 일부 모델이 일시적으로 사용되지 않을 수 있습니다.
- API 키는 저장되지 않으므로 실행할 때마다 다시 입력해야 합니다.
